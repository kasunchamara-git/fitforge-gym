# FitForge Gym – Sports Center

A responsive promotional website for a local fitness brand built with Next.js.

## Tech Stack
- Next.js
- React
- CSS (Custom Properties)
- Google Fonts (Bebas Neue, Barlow)

## Setup Instructions
1. Clone the repository
   git clone https://github.com/kasunchamara-git/fitforge-gym.git
2. Install dependencies
   npm install
3. Run the development server
   npm run dev
4. Open http://localhost:3000
